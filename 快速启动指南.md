# 🚀 快速启动指南

**新版设计特点**：全新蓝绿科技风格，更现代、更立体的视觉效果！

## 第一步：安装依赖

在项目根目录（发卡网文件夹）打开终端，运行：

```bash
npm install
```

这会自动安装所有需要的包（Express、SQLite、JWT等）。

## 第二步：配置环境变量

复制环境变量模板文件：

```bash
cp .env.example .env
```

然后编辑 `.env` 文件，**至少修改以下配置**：

```bash
# JWT 密钥（重要！改成任意随机字符串）
JWT_SECRET=请修改为你的随机密钥123456abcdef

# 管理员账号
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=admin123456
```

**注意**：支付宝和微信支付的配置可以暂时不改，系统会自动使用测试模式。

## 第三步：初始化数据库

运行数据库初始化脚本：

```bash
npm run init-db
```

你会看到类似输出：
```
✅ 创建数据库目录
✅ 连接到 SQLite 数据库
✅ 用户表创建成功
✅ 商品表创建成功
✅ 卡密表创建成功
✅ 订单表创建成功
✅ 管理员账号创建成功
   邮箱: admin@example.com
   密码: admin123456
✅ 示例商品数据插入成功
🎉 数据库初始化完成！
```

## 第四步：启动服务器

```bash
npm start
```

或者使用开发模式（文件修改后自动重启）：

```bash
npm run dev
```

看到这个输出说明成功了：

```
🚀 服务器启动成功！

📍 访问地址:
   前台商城: http://localhost:3000
   管理后台: http://localhost:3000/admin

⚙️  环境: development

💡 提示: 使用 Ctrl+C 停止服务器
```

## 第五步：访问系统

### 1. 前台商城
打开浏览器访问：http://localhost:3000

**测试购买流程：**
1. 点击"登录/注册"，注册一个普通用户账号
2. 登录后选择商品，点击"立即购买"
3. 选择支付方式（会自动进入测试模式）
4. 点击确定后跳转到支付成功页面，自动获取卡密

### 2. 管理后台
打开浏览器访问：http://localhost:3000/admin

**默认管理员账号：**
- 邮箱：admin@example.com
- 密码：admin123456

**管理员功能：**
- ✅ 查看销售统计
- ✅ 添加/删除商品
- ✅ 为商品添加卡密
- ✅ 查看所有订单

## 📝 添加卡密的方法

1. 登录管理后台
2. 找到商品列表，点击"添加卡密"
3. 按照格式输入卡密（每行一个）：

```
卡号1|密码1
卡号2|密码2
卡号3
```

示例：
```
ChatGPT-Account-001|Password123
ChatGPT-Account-002|Password456
ChatGPT-Account-003|Password789
```

## 🌐 部署到线上（可选）

### 方案一：使用 Railway（推荐）

1. 访问 https://railway.app
2. 用 GitHub 登录
3. 新建项目，选择"Deploy from GitHub repo"
4. 选择你的仓库
5. 添加环境变量（复制 .env 内容）
6. 自动部署，获得在线网址

### 方案二：使用 Render

1. 访问 https://render.com
2. 新建 Web Service
3. 连接 GitHub 仓库
4. 设置启动命令：`npm start`
5. 添加环境变量
6. 部署完成

### 方案三：阿里云/腾讯云

1. 购买云服务器
2. 安装 Node.js
3. 上传代码
4. 运行 `npm install` 和 `npm start`
5. 配置 Nginx 反向代理

## ⚠️ 注意事项

### 安全提醒
- ✅ 生产环境务必修改 `JWT_SECRET` 为复杂随机字符串
- ✅ 修改默认管理员密码
- ✅ 定期备份 `database/shop.db` 文件
- ✅ 不要将 `.env` 文件提交到 Git

### 支付配置
- 测试模式：点击购买后会模拟支付成功
- 真实支付：需要在 `.env` 中配置支付宝/微信商户信息
- 申请支付宝：https://open.alipay.com
- 申请微信支付：https://pay.weixin.qq.com

## 🐛 常见问题

### 1. 安装依赖失败
```bash
# 使用国内镜像源
npm install --registry=https://registry.npmmirror.com
```

### 2. 端口被占用
修改 `.env` 文件中的 `PORT` 值：
```
PORT=3001
```

### 3. 数据库错误
删除 `database` 文件夹，重新运行 `npm run init-db`

### 4. 找不到商品
确保已经在管理后台为商品添加了卡密，否则会显示"售罄"
