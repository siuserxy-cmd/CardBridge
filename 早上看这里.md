# 🌅 早上好！你的发卡网站已经准备好了

## 🚀 快速启动（3步）

### 方法一：双击运行（推荐）

**Mac/Linux 用户**：
1. 双击 `启动.sh` 文件
2. 如果提示权限问题，在终端运行：
   ```bash
   chmod +x 启动.sh
   ./启动.sh
   ```

**Windows 用户**：
1. 双击 `启动.bat` 文件

### 方法二：命令行运行

在项目文件夹打开终端，依次运行：

```bash
# 1. 安装依赖（首次运行）
npm install

# 2. 初始化数据库（首次运行）
npm run init-db

# 3. 添加示例卡密（首次运行，可选）
npm run add-cards

# 4. 启动服务器
npm start
```

## 🌐 访问网站

启动成功后，打开浏览器访问：

- **前台商城**：http://localhost:3000
- **管理后台**：http://localhost:3000/admin

## 🎮 快速测试

### 1. 测试购买流程（前台）
1. 访问 http://localhost:3000
2. 点击"登录/注册"，注册一个新账号
3. 选择商品，点击"立即购买"
4. 选择支付方式（会自动进入测试模式）
5. 确认后会模拟支付成功，自动获取卡密

### 2. 管理后台功能
1. 访问 http://localhost:3000/admin
2. 用管理员账号登录
3. 可以：
   - 查看销售统计
   - 添加/删除商品
   - 批量添加卡密
   - 查看所有订单

## 🐛 遇到问题？

### 端口被占用
修改 `.env` 文件中的 `PORT=3000` 改为其他端口

### 依赖安装失败
使用国内镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```

### 数据库错误
删除 `database` 文件夹，重新运行 `npm run init-db`
