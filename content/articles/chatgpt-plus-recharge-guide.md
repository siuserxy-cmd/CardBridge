# ChatGPT Plus 代充完整教程 - 3 步完成订阅

很多用户第一次来本站都会问：**"ChatGPT Plus 怎么充？需要什么？安全吗？"**

这篇教程用 3 个步骤完整演示从下单到到账的全流程，读完 3 分钟就能上手。

## 你需要准备什么

- 一个 **已注册的 ChatGPT 账号**（免费账号即可，我们不提供注册服务）
- 账号对应的 **邮箱地址**
- 账号的 **Access Token**（临时会话令牌，不是密码）
- 支付宝或微信支付

**重要**：我们不需要你的 ChatGPT 登录密码，也不需要你的邮箱密码。全程只使用一次性的 Session Token。

## 第一步：获取 Access Token

1. 打开浏览器，访问 [chat.openai.com](https://chat.openai.com) 并登录你的账号
2. 登录后，在同一个浏览器打开一个新标签页，访问 `https://chat.openai.com/api/auth/session`
3. 页面会返回一段 JSON，找到 `"accessToken"` 字段，复制引号里的那一长串字符
4. 这段字符就是你的 Access Token，一般以 `eyJ...` 开头

这个 Token 有 **临时有效期**，大约几小时，所以建议在准备下单前再去获取，避免过期。

## 第二步：下单并提交 Token

1. 回到本站首页，选择 **"ChatGPT Plus 自动充值"** 类商品
2. 点击 **"立即购买"**，按提示填写：
   - 接收邮箱（用于发送充值结果）
   - ChatGPT 账号邮箱（你要充值的那个账号）
   - Access Token（刚才复制的那段字符）
3. 选择支付方式：**支付宝** 或 **微信**
4. 点击确认，跳转到支付二维码页面

## 第三步：扫码支付，自动到账

1. 用手机打开支付宝或微信，扫描二维码完成支付
2. 付款成功后，页面会自动刷新到订单详情
3. 系统会在后台自动处理充值，**通常 1 分钟内完成**
4. 充值成功后你会收到一封邮件通知，同时 ChatGPT 账号会自动升级为 Plus
