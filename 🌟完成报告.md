## 🌅 明早起来怎么做？

### 最快方式（推荐）

**双击运行**：
- Mac：双击 `启动.sh`
- Windows：双击 `启动.bat`

### 命令行方式

```bash
# 1. 进入项目文件夹
cd 发卡网

# 2. 安装依赖（首次）
npm install

# 3. 初始化数据库（首次）
npm run init-db

# 4. 添加示例卡密（可选）
npm run add-cards

# 5. 启动服务器
npm start
```

### 访问网站

- 前台：http://localhost:3000
- 后台：http://localhost:3000/admin

### 默认账号

- 邮箱：admin@example.com
- 密码：admin123456

## 🔐 安全提醒

### 立即修改（重要）

1. **JWT 密钥**
   - 打开 `.env` 文件
   - 修改 `JWT_SECRET` 为随机字符串

2. **管理员密码**
   - 首次登录后立即修改
   - 或修改 `.env` 中的 `ADMIN_PASSWORD`

3. **支付配置**
   - 当前为测试模式
   - 真实支付需配置商户信息

### 部署时必做

- ✅ 修改所有默认密码
- ✅ 配置环境变量
- ✅ 定期备份数据库
